# 🎵 YouVibe

YouVibe is a modern full-stack music streaming platform built to provide an immersive and interactive music experience. Users can discover and stream their favorite songs, create personalized playlists and albums, enjoy seamless loop playback, and connect with other active listeners through real-time chat.

Designed with scalability and performance in mind, YouVibe combines a clean user interface with powerful backend functionality, making it a complete music streaming solution.

---

## ✨ Features

### 🎧 Music Streaming
- Stream songs with a smooth and responsive music player
- Play, pause, skip, and seek tracks
- Loop songs for continuous playback
- Browse music by albums and artists

### 📂 Playlists & Albums
- Create and manage personalized playlists
- Create and organize albums
- Add or remove songs anytime

### 💬 Real-Time Chat
- Chat with active listeners in real time
- Instant messaging powered by Socket.IO
- Connect with music lovers around the world

### 🔐 Authentication
- Secure user authentication
- Google Sign-In
- Protected routes and secure user sessions

### 👨‍💼 Admin Dashboard
- Add, update, and delete songs
- Manage albums and artists
- Easily control platform content
